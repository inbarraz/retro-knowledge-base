## Platform-Specific Knowledge

### MD-B12-60 Ultra High Resolution Video Card

#### Overview
- ISA bus video card from 1987 - Source: [UaJ3qXZSpwc]
- 1280x1024 resolution at 256 colors from palette of 16.7 million - Source: [UaJ3qXZSpwc]
- Designed for CAD/CAM workstations - Source: [UaJ3qXZSpwc]
- Original price extremely high (thousands of dollars) - Source: [UaJ3qXZSpwc]
- Required special high-resolution monitors - Source: [UaJ3qXZSpwc]

#### Graphics Processor
- Texas Instruments TMS34010 32-bit graphics processor - Source: [UaJ3qXZSpwc]
- TMS34010 runs at 6 MIPS - Source: [UaJ3qXZSpwc]
- Dedicated graphics processor offloads CPU - Source: [UaJ3qXZSpwc]
- Can draw lines, circles, fills in hardware - Source: [UaJ3qXZSpwc]

#### RAMDAC
- Brooktree BT458 RAMDAC - Source: [UaJ3qXZSpwc]
- Handles digital-to-analog conversion for video output - Source: [UaJ3qXZSpwc]
- 256 color palette registers - Source: [UaJ3qXZSpwc]
- Each palette entry selects from 16.7 million colors - Source: [UaJ3qXZSpwc]

#### VGA Pass-Through Feature
- Has VGA feature connector input - Source: [UaJ3qXZSpwc]
- Can overlay/combine with VGA card output - Source: [UaJ3qXZSpwc]
- Allows running DOS games on VGA while using high-res for Windows - Source: [UaJ3qXZSpwc]
- Feature connector is 26-pin IDC header - Source: [UaJ3qXZSpwc]
- Same connector used on VGA cards for video overlay devices - Source: [UaJ3qXZSpwc]

#### Memory
- Uses VSIM memory modules (proprietary EIZO format) - Source: [UaJ3qXZSpwc]
- 1MB standard, expandable with additional VSIMs - Source: [UaJ3qXZSpwc]
- More VRAM = more colors at high resolution - Source: [UaJ3qXZSpwc]
- VSIMs rare and hard to find - Source: [UaJ3qXZSpwc]

#### Emulation Modes
- Can emulate CGA in 640x200 mode - Source: [UaJ3qXZSpwc]
- Can emulate monochrome Hercules/MDA - Source: [UaJ3qXZSpwc]
- Emulation modes useful for DOS software compatibility - Source: [UaJ3qXZSpwc]
- Software switch between modes - Source: [UaJ3qXZSpwc]

#### Driver Support
- Windows 3.0/3.1 drivers available - Source: [UaJ3qXZSpwc]
- AutoCAD specifically supported - Source: [UaJ3qXZSpwc]
- GEM Desktop had drivers - Source: [UaJ3qXZSpwc]
- TIGA (Texas Instruments Graphics Architecture) standard - Source: [UaJ3qXZSpwc]

#### Physical
- Full-length ISA card - Source: [UaJ3qXZSpwc]
- Very long, requires deep case - Source: [UaJ3qXZSpwc]
- DB15 video output connector - Source: [UaJ3qXZSpwc]
- Some versions have BNC connectors for RGB output - Source: [UaJ3qXZSpwc]
