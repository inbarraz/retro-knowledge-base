## Platform-Specific Knowledge

### Coleco Adam

#### Overview
- Home computer system from Coleco (1983) - Source: [j-8yycf3ob4]
- Expansion module that started as ColecoVision add-on - Source: [j-8yycf3ob4]
- Became standalone computer system - Source: [j-8yycf3ob4]
- Complete system includes: computer module, keyboard, printer, two digital data packs - Source: [j-8yycf3ob4]

#### Specifications
- Z80 CPU - Source: [j-8yycf3ob4]
- 64KB DRAM + 16KB VRAM = "80K" total (marketing claim) - Source: [j-8yycf3ob4]
- TMS9928A or TMS99928 graphics chip (same as TI-99/4A, MSX, Sega Master System) - Source: [j-8yycf3ob4]
- SN76498 sound chip (same family as TI-99/4A, Sega, BBC Micro) - Source: [j-8yycf3ob4]
- Came bundled with Buck Rogers game cartridge - Source: [j-8yycf3ob4]

#### Critical Power System Design
- **PRINTER CONTAINS THE POWER SUPPLY** - Source: [j-8yycf3ob4]
- System will not operate without printer connected - Source: [j-8yycf3ob4]
- Printer is required even if you never print anything - Source: [j-8yycf3ob4]
- Power flows from printer to rest of system - Source: [j-8yycf3ob4]
- This is the most unusual/infamous aspect of Adam design - Source: [j-8yycf3ob4]

#### Digital Data Pack (High-Speed Tape)
- Two data pack drives included with system - Source: [j-8yycf3ob4]
- NOT standard audio cassette - uses high-speed digital encoding - Source: [j-8yycf3ob4]
- Much faster than typical home computer tape storage - Source: [j-8yycf3ob4]
- Proprietary format - requires Coleco data pack tapes - Source: [j-8yycf3ob4]
- Data packs were included with system - Source: [j-8yycf3ob4]

#### Expansion
- Expansion port on back for accessories - Source: [j-8yycf3ob4]
- Could add floppy drives (5.25") - Source: [j-8yycf3ob4]
- RAM expansion available - Source: [j-8yycf3ob4]
- FujiNet compatible (modern WiFi networking device) - Source: [j-8yycf3ob4]

#### Keyboard
- Full-sized keyboard included - Source: [j-8yycf3ob4]
- Connects to main computer unit - Source: [j-8yycf3ob4]
- Had reputation for decent typing experience - Source: [j-8yycf3ob4]

#### Software
- SmartBasic included - Source: [j-8yycf3ob4]
- SmartWriter word processor built-in - Source: [j-8yycf3ob4]
- CP/M available (Z80 compatible) - Source: [j-8yycf3ob4]
- ColecoVision game compatibility - Source: [j-8yycf3ob4]

#### Packaging and Collectibility
- Complete systems with original packaging are rare - Source: [j-8yycf3ob4]
- Original Styrofoam inserts valuable for collectors - Source: [j-8yycf3ob4]
- System came in large box due to multiple components - Source: [j-8yycf3ob4]
- Finding complete system with both data packs, printer, keyboard is uncommon - Source: [j-8yycf3ob4]

#### Historical Context
- Released 1983, discontinued 1985 - Source: [j-8yycf3ob4]
- Part of home computer market crash era - Source: [j-8yycf3ob4]
- Coleco better known for ColecoVision game console - Source: [j-8yycf3ob4]
- Adam was attempt to compete with Commodore 64, Apple II - Source: [j-8yycf3ob4]

## Troubleshooting

### Power Issues
- System won't turn on: Verify printer is connected (contains PSU) - Source: [j-8yycf3ob4]
- Check printer power connection first before other diagnosis - Source: [j-8yycf3ob4]

### Data Pack Issues
- Ensure using Coleco digital data packs, not audio cassettes - Source: [j-8yycf3ob4]
- Clean tape heads if read errors occur - Source: [j-8yycf3ob4]
