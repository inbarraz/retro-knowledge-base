## Platform-Specific Knowledge

### KC 85 Series (Kleincomputer)

#### Overview
- East German home computer series from the 1980s - Source: [At9UNYFHuaE]
- "Kleincomputer" means "compact computer" - Source: [At9UNYFHuaE]
- Originally called HC 900 (Heimcomputer/home computer) - renamed to avoid "games" association - Source: [At9UNYFHuaE]
- Made by RFT (Rundfunk- und Fernmelde-Technik) - Source: [At9UNYFHuaE]
- Very rare outside Germany, especially in North America - Source: [At9UNYFHuaE]

#### KC 85/3 Specifications
- CPU: U880 (Soviet Z80 clone) with 2.5mm metric pin spacing vs 2.54mm - Source: [At9UNYFHuaE]
- 32KB RAM (using 4K x 1-bit DRAM chips in multiple banks) - Source: [At9UNYFHuaE]
- 16KB ROM - Source: [At9UNYFHuaE]
- Graphics: 320x256 resolution - Source: [At9UNYFHuaE]
- Built-in BASIC interpreter - Source: [At9UNYFHuaE]
- RF modulator output - Source: [At9UNYFHuaE]
- RGB output - Source: [At9UNYFHuaE]

#### Power Requirements
- 220V 50Hz, 25 watts - Source: [At9UNYFHuaE]
- Internal linear power supply (not switching) - Source: [At9UNYFHuaE]
- Requires +12V, +5V, -5V rails - Source: [At9UNYFHuaE]
- Can be replaced with Pico ATX supply with -5V adapter - Source: [At9UNYFHuaE]
- Test PSU option available (110-240V compatible) - Source: [At9UNYFHuaE]

#### Expansion System
- Front-loading expansion modules - Source: [At9UNYFHuaE]
- Metal latches hold modules in place - Source: [At9UNYFHuaE]
- 64KB RAM expansion module available - Source: [At9UNYFHuaE]
- Card edge connector interface - Source: [At9UNYFHuaE]
- Can be stacked with backplane for more slots - Source: [At9UNYFHuaE]
- Available modules: V24/V25 serial, EPROM writers, floppy interfaces - Source: [At9UNYFHuaE]

#### Keyboard
- Separate unit with TRS-style single jack connection - Source: [At9UNYFHuaE]
- Contains electronics (U807D chip, CD4011 analog switch) - Source: [At9UNYFHuaE]
- Powered through single cable, transmits keypresses back - Source: [At9UNYFHuaE]
- QWERTZ layout (German) with swapped Y/Z - Source: [At9UNYFHuaE]
- Rubber dome switches, calculator-style feel - Source: [At9UNYFHuaE]

#### KC 85 Series Evolution
- KC 85/1 and 85/2: 16KB RAM, 4KB ROM - Source: [At9UNYFHuaE]
- KC 85/3: 32KB RAM, 16KB ROM - Source: [At9UNYFHuaE]
- KC 85/4: Similar improvements in ROM/memory mapping - Source: [At9UNYFHuaE]
- KC 85/5: Planned but cancelled when Berlin Wall fell - Source: [At9UNYFHuaE]

#### BASIC Commands
- CS (clear screen), SE (load), C (save), LIST, LINES - Source: [At9UNYFHuaE]
- GOSUB, RETURN, ON GOTO, ON GOSUB - Source: [At9UNYFHuaE]
- Built-in sound commands - Source: [At9UNYFHuaE]
- VPOKE/VPEEK for video memory - Source: [At9UNYFHuaE]
- TRACE ON/OFF for debugging - Source: [At9UNYFHuaE]
- SWITCH, RENUM commands - Source: [At9UNYFHuaE]

#### Tape Storage
- Uses standard compact cassettes - Source: [At9UNYFHuaE]
- Data Cord GCC 6025 tape recorder (RFT branded) - Source: [At9UNYFHuaE]
- AC synchronous motor - requires 50Hz (60Hz plays at wrong speed) - Source: [At9UNYFHuaE]
- DIN connector for computer interface - Source: [At9UNYFHuaE]
- DK (digital cassette) type tapes available - Source: [At9UNYFHuaE]

#### Construction Quality
- Metal top cover - Source: [At9UNYFHuaE]
- Large bypass capacitors throughout - Source: [At9UNYFHuaE]
- Very overbuilt compared to Western computers - Source: [At9UNYFHuaE]

#### Community
- Active enthusiast scene (mostly German language) - Source: [At9UNYFHuaE]
- Modern add-ons and fixes available - Source: [At9UNYFHuaE]
- KC 85/5 can be built from scratch or modified from older models - Source: [At9UNYFHuaE]
- Emulators available (floooh.github.io/virtualkc) - Source: [At9UNYFHuaE]

### GDR Computer Industry

#### Other GDR Computers

- PC 1715: Z80-based, 2KB ROM, loaded from floppy - Source: [At9UNYFHuaE]
- A 7100: 8086-based, 256KB RAM, 640x400 resolution (not PC compatible) - Source: [At9UNYFHuaE]

- Surprisingly frequent use of English text on products - Source: [At9UNYFHuaE]
