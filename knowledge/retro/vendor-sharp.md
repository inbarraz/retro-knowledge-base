## Platform-Specific Knowledge

### Sharp Zaurus ZR-3000

#### Overview
- Mid-1990s palmtop computer - Source: [rDAv6K2i2tk]
- Made in Japan - Source: [rDAv6K2i2tk]
- Z80-based system-on-chip (not x86) - Source: [rDAv6K2i2tk]
- Runs proprietary Sharp OS (not DOS) - Source: [rDAv6K2i2tk]
- Touchscreen with stylus input - Source: [rDAv6K2i2tk]
- Small physical keyboard - Source: [rDAv6K2i2tk]
- No backlight on display - Source: [rDAv6K2i2tk]
- Reflective plastic screen (hard to see in poor lighting) - Source: [rDAv6K2i2tk]

#### Power Requirements
- Two AA batteries for main power - Source: [rDAv6K2i2tk]
- Two CR2032 batteries for memory backup (volatile RAM) - Source: [rDAv6K2i2tk]
- AC adapter input available - Source: [rDAv6K2i2tk]

#### Memory
- 614K available memory - Source: [rDAv6K2i2tk]
- Volatile RAM (requires battery backup) - Source: [rDAv6K2i2tk]
- No internal card slot on ZR-3000 (ZR-5000 has card slot) - Source: [rDAv6K2i2tk]

#### Display
- High resolution LCD with grayscale capability - Source: [rDAv6K2i2tk]
- Shaded UI elements (dithered grays) - Source: [rDAv6K2i2tk]
- Screen opens 180 degrees flat - Source: [rDAv6K2i2tk]
- Palm Pilot-like touchscreen interface - Source: [rDAv6K2i2tk]

#### Built-in Software
- Word processor (MS Word compatible) - Source: [rDAv6K2i2tk]
- Spreadsheet (MS Excel compatible) - Source: [rDAv6K2i2tk]
- Calculator (floating window) - Source: [rDAv6K2i2tk]
- Contacts/Address book - Source: [rDAv6K2i2tk]
- Notes (with drawing capability) - Source: [rDAv6K2i2tk]
- World clock - Source: [rDAv6K2i2tk]
- Scrapbook with clip art (buildings, roads, vehicles, etc.) - Source: [rDAv6K2i2tk]
- Terminal emulator (VT-100 compatible) - Source: [rDAv6K2i2tk]

#### Terminal Emulator Capabilities
- Baud rates up to 14400 - Source: [rDAv6K2i2tk]
- VT-100 terminal emulation - Source: [rDAv6K2i2tk]
- 80-column display mode (very small font) - Source: [rDAv6K2i2tk]
- Local echo option - Source: [rDAv6K2i2tk]
- Software flow control - Source: [rDAv6K2i2tk]
- Terminal response is laggy when typing fast - Source: [rDAv6K2i2tk]
- Word processor feels faster than terminal - Source: [rDAv6K2i2tk]

#### Connectivity
- Serial port via proprietary connector - Source: [rDAv6K2i2tk]
- Serial cable included for PC sync - Source: [rDAv6K2i2tk]
- Built-in IR (infrared) for wireless printing - Source: [rDAv6K2i2tk]
- Windows 95 sync software available - Source: [rDAv6K2i2tk]

#### Spreadsheet Features
- Drag-to-select cells - Source: [rDAv6K2i2tk]
- Formula support (no equals sign needed) - Source: [rDAv6K2i2tk]
- Displays "recalculating" during computation - Source: [rDAv6K2i2tk]
- Can turn off auto-recalculation for large spreadsheets - Source: [rDAv6K2i2tk]
- Shows results in exponent notation for large numbers - Source: [rDAv6K2i2tk]

#### Drawing/Notes
- Object-oriented drawing in notes - Source: [rDAv6K2i2tk]
- Can move objects around after creation - Source: [rDAv6K2i2tk]
- Cut/paste for selections - Source: [rDAv6K2i2tk]

- Built-in clip art (buildings, roads, vehicles, cakes, airports) - Source: [rDAv6K2i2tk]
- Picture editor not accessible outside startup screen customization - Source: [rDAv6K2i2tk]

#### User Interface
- Graphical windowed interface - Source: [rDAv6K2i2tk]
- Double-click to open apps - Source: [rDAv6K2i2tk]

#### ZR-5000 vs ZR-3000

- Card slot supports SRAM cards (1-2MB, battery-backed) - Source: [rDAv6K2i2tk]
- Card slot supports non-ATA flash cards (up to 16MB) - Source: [rDAv6K2i2tk]
- Can allocate 2MB for programs on card - Source: [rDAv6K2i2tk]
