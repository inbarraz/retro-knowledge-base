## Platform-Specific Knowledge

### Microsoft MACH-20 286 Accelerator

#### Overview
- Accelerator card that upgrades PC/XT to 286 performance - Source: [a4Py5VnKQE8]
- Runs at 8MHz (uses 16MHz crystal divided by 2) - Source: [a4Py5VnKQE8]
- Name suggests 20MHz but actual speed is 8MHz - Source: [a4Py5VnKQE8]
- Designed for DOS 2.0 or higher - Source: [a4Py5VnKQE8]
- Designed for OS/2 and Windows applications - Source: [a4Py5VnKQE8]

#### Hardware Components
- Siemens 80286 processor - Source: [a4Py5VnKQE8]
- 16KB 16-bit wide cache memory for faster data access - Source: [a4Py5VnKQE8]
- Socket for 80287 math coprocessor - Source: [a4Py5VnKQE8]
- Up to 3.5MB high-speed RAM expansion - Source: [a4Py5VnKQE8]
- Microsoft bus mouse connector - Source: [a4Py5VnKQE8]
- RAM daughter board plugs on top of main card - Source: [a4Py5VnKQE8]

#### Disk Plus Floppy Controller
- Companion card that attaches to MACH-20 - Source: [a4Py5VnKQE8]
- Supports up to 1.44MB high-density floppy drives - Source: [a4Py5VnKQE8]
- Uses Western Digital floppy controller IC - Source: [a4Py5VnKQE8]
- Has onboard BIOS ROM labeled "DISK PLUS" - Source: [a4Py5VnKQE8]
- Copyright 1987 Microsoft Corp - Source: [a4Py5VnKQE8]

#### Physical Assembly
- Three-card assembly when fully populated - Source: [a4Py5VnKQE8]
- Main MACH-20 card at bottom - Source: [a4Py5VnKQE8]
- RAM expansion board connects via edge connector and standoffs - Source: [a4Py5VnKQE8]
- Disk Plus floppy controller screws on top - Source: [a4Py5VnKQE8]
- Metal standoffs hold cards together - Source: [a4Py5VnKQE8]
- Full-length ISA card when assembled - Source: [a4Py5VnKQE8]

#### Installation
- 40-pin ribbon cable connects from card to CPU socket on XT motherboard - Source: [a4Py5VnKQE8]
- "Blanking plug" required for 80287 socket on motherboard - Source: [a4Py5VnKQE8]
- Original CPU removed from motherboard socket - Source: [a4Py5VnKQE8]
- Card provides all CPU functions via ribbon cable - Source: [a4Py5VnKQE8]

#### RAM Configuration
- RAM board uses Intel 21256 chips - Source: [a4Py5VnKQE8]
- Also TI TMS4256-10 (100ns) chips - Source: [a4Py5VnKQE8]
- MT (Micron) 1259-12 256KB chips also used - Source: [a4Py5VnKQE8]
- 1.5MB populated, 2MB maximum capacity - Source: [a4Py5VnKQE8]
- RAM board marked "PCP-2" Part 00022 Rev 1.0 - Source: [a4Py5VnKQE8]

#### Documentation Status
- Original user manual has not been found/scanned - Source: [a4Py5VnKQE8]
- Only Microsoft TechNet articles survive online - Source: [a4Py5VnKQE8]
- TechNet has updated drivers available - Source: [a4Py5VnKQE8]
- One-page marketing sheet found online - Source: [a4Py5VnKQE8]
- Original engineer at Microsoft did not keep documentation - Source: [a4Py5VnKQE8]

#### Historical Notes
- Designed 1987 - Source: [a4Py5VnKQE8]
- Made your PC/XT perform "like an AT" - Source: [a4Py5VnKQE8]
- Graphics and windowing applications were target use case - Source: [a4Py5VnKQE8]
- South Bay Users Group reviewed it in 1986 - Source: [a4Py5VnKQE8]
- Mentioned having "rather lengthy user manual" - Source: [a4Py5VnKQE8]
