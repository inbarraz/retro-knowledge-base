## Platform-Specific Knowledge

### Megalith Plus Memory Expansion Card

#### Overview
- 8MB ISA RAM expansion card - Source: [nn76UCUIo3Q]
- Provides EMS and XMS memory for 286/386 systems - Source: [nn76UCUIo3Q]
- Alternative to motherboard RAM upgrades - Source: [nn76UCUIo3Q]

#### EMS/XMS Memory
- EMS 4.0 (Expanded Memory Specification) compatible - Source: [nn76UCUIo3Q]
- Provides expanded memory for DOS applications - Source: [nn76UCUIo3Q]
- Driver required to expose memory as XMS (extended) - Source: [nn76UCUIo3Q]
- Useful for DOS games requiring EMS - Source: [nn76UCUIo3Q]

#### System Compatibility
- Works with 286 systems - Source: [nn76UCUIo3Q]
- Works with 386SX and 386DX systems - Source: [nn76UCUIo3Q]
- May work with 486 but less common usage - Source: [nn76UCUIo3Q]
- Requires available 16-bit ISA slot - Source: [nn76UCUIo3Q]

#### Use Cases
- Systems with limited motherboard RAM capacity - Source: [nn76UCUIo3Q]
- When compatible SIMMs are expensive/unavailable - Source: [nn76UCUIo3Q]
- DOS gaming requiring EMS memory - Source: [nn76UCUIo3Q]
- Lotus 1-2-3 and other EMS-aware applications - Source: [nn76UCUIo3Q]

#### Configuration
- DIP switches or jumpers for memory base address - Source: [nn76UCUIo3Q]
- Software driver loaded in CONFIG.SYS - Source: [nn76UCUIo3Q]
- May require specific memory manager configuration - Source: [nn76UCUIo3Q]
